# איך להוסיף תמונות אמיתיות ליעדים ומדינות

## דרך מהירה - סקריפט SQL

יצרתי 2 סקריפטים שמעדכנים תמונות אמיתיות מ-Unsplash:

### 1. תמונות ליעדים
**קובץ:** `scripts/add-real-images-to-destinations.sql`

הסקריפט מעדכן תמונות ליעדים נפוצים:
- בנגקוק, פוקט, ציאנג מאי (תאילנד)
- אתונה, סנטוריני, מיקונוס (יוון)
- רומא, ונציה, מילאנו, פירנצה (איטליה)
- מדריד, ברצלונה, סביליה (ספרד)
- איסטנבול, קפדוקיה, אנטליה (טורקיה)
- דובאי, אבו דאבי (דובאי)
- תל אביב, ירושלים, אילת (ישראל)

### 2. תמונות למדינות
**קובץ:** `scripts/add-real-images-to-countries.sql`

הסקריפט מעדכן תמונות למדינות נפוצות.

---

## איך להריץ

1. פתח את Supabase → SQL Editor
2. פתח את הקובץ `scripts/add-real-images-to-destinations.sql`
3. העתק את כל התוכן
4. הדבק ב-SQL Editor
5. לחץ **Run**

**הערה:** הסקריפטים כבר מכילים את מזהה הסוכנות שלך (`4af931ea-e4f8-4728-9d82-30de9b2f2b14`), אז אתה יכול להריץ אותם ישירות!

---

## איך זה עובד

הסקריפטים משתמשים ב-URLs של Unsplash - שירות תמונות חינמי ואיכותי.

**יתרונות:**
- ✅ תמונות איכותיות ומקצועיות
- ✅ חינמי לחלוטין
- ✅ לא דורש API key
- ✅ תמונות מתאימות לכל יעד

**הערה:** התמונות נטענות מ-Unsplash, אז צריך חיבור לאינטרנט כדי לראות אותן.

---

## עדכון תמונות מותאמות אישית

אם אתה רוצה תמונות ספציפיות יותר:

1. **מצא תמונה ב-Unsplash:**
   - לך ל-https://unsplash.com
   - חפש את היעד (לדוגמה: "Bangkok")
   - לחץ על התמונה
   - לחץ על "Download" או "Copy link"
   - העתק את ה-URL

2. **עדכן את הסקריפט:**
   - פתח את `scripts/add-real-images-to-destinations.sql`
   - מצא את היעד שאתה רוצה לעדכן
   - החלף את ה-URL ב-`image_url = '...'`

3. **הרץ את הסקריפט שוב**

---

## בדיקה

אחרי הרצת הסקריפט, בדוק:

```sql
-- בדוק יעדים עם תמונות
SELECT name, image_url 
FROM destinations 
WHERE agency_id = '4af931ea-e4f8-4728-9d82-30de9b2f2b14' 
  AND image_url IS NOT NULL;

-- בדוק מדינות עם תמונות
SELECT name, image_url 
FROM countries 
WHERE agency_id = '4af931ea-e4f8-4728-9d82-30de9b2f2b14' 
  AND image_url IS NOT NULL;
```
